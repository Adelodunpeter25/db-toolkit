# DB Toolkit

A native desktop application with a modern UI built entirely in PySide6 + QML, providing a seamless, cross-platform experience for developers who want database exploration, query execution, and migration management in a single tool.

## Features

- **Multi-Database Support**: PostgreSQL, MySQL, SQLite, and MongoDB
- **Database Explorer**: Browse schemas, tables, views, columns, and collections
- **Query Editor**: SQL editor for relational databases and MongoDB query editor
- **Data Management**: Inline row editing and CSV import/export
- **Connection Manager**: Manage multiple local and remote database connections
- **Migration Tools**: Database migration and schema management

## Requirements

- Python 3.11+
- PySide6

## Installation

```bash
uv install
```

## Usage

```bash
dbtoolkit
```
